## nrm

1. 安装
npm install nrm -g

2. 查看可用地址
nrm ls

3. 切换到淘宝试试
nrm use taobao

4. 添加我们内部搭建的私有包源，取名strong
nrm add strong http://115.29.205.204:10001/

5. 切换
nrm use strong
